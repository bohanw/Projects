----------------------------------------------------------------------
ALL_CONV                                                        (Conv)
----------------------------------------------------------------------
ALL_CONV : conv

SYNOPSIS
Conversion that always succeeds, raising the {UNCHANGED} exception.

KEYWORDS
conversion, identity.

DESCRIBE
When applied to a term {t}, the conversion {ALL_CONV} raises the
special {UNCHANGED} exception.

FAILURE
Never fails.

USES
Identity element for {THENC}.

SEEALSO
Conv.NO_CONV, Thm.REFL.
